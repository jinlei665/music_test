> 日期: 2026-04-16
> 状态: 已批准
> 阶段: 首页 + 登录注册页

---

## 1. 概述

重构音乐网站的前端页面，采用蔚蓝色清新渐变风格，结合 Three.js 3D 粒子效果和毛玻璃 UI。

**设计风格：**
- 配色：清新渐变（浅蓝到深蓝渐变，如海洋/天空）
- 粒子：Three.js 3D 粒子，鼠标移动时粒子响应
- 导航栏：毛玻璃效果（backdrop-filter: blur）
- 过渡动画：淡入淡出
- 按钮/卡片动画：弹性效果（bounce 感）

## 3. 设计规范

### 3.1 颜色系统

| 用途 | 颜色 | 说明 |
|------|------|------|
| 主色-浅蓝 | `#87CEEB` | Sky Blue，浅色元素 |
| 主色-深蓝 | `#1E3A5F` | 深蓝色，渐变终点 |
| 主色-中蓝 | `#4A90D9` | 中蓝色，过渡色 |
| 强调色 | `#00D4FF` | 霓虹蓝，发光效果 |
| 背景色 | `#0A1628` | 深蓝黑，主背景 |
| 卡片背景 | `rgba(255,255,255,0.1)` | 半透明白色 |
| 文字主色 | `#FFFFFF` | 白色文字 |
| 文字次色 | `rgba(255,255,255,0.7)` | 半透明白色 |
| 边框色 | `rgba(255,255,255,0.2)` | 半透明白边框 |

### 3.2 渐变

```css
/* 背景渐变 */
background: linear-gradient(135deg, #0A1628 0%, #1E3A5F 50%, #0A1628 100%);

/* 按钮渐变 */
background: linear-gradient(90deg, #4A90D9, #00D4FF);

/* 卡片渐变边框 */
border: 1px solid transparent;
background: linear-gradient(#0A1628, #0A1628) padding-box,
            linear-gradient(135deg, #4A90D9, #00D4FF) border-box;
```

### 3.3 毛玻璃效果

```css
.glass {
    background: rgba(255, 255, 255, 0.1);
    backdrop-filter: blur(20px);
    -webkit-backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.2);
    border-radius: 16px;
}
```

### 3.4 阴影

```css
/* 发光效果 */
box-shadow: 0 0 20px rgba(0, 212, 255, 0.3);

/* 卡片阴影 */
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
```

### 3.5 圆角

| 元素 | 圆角 |
|------|------|
| 卡片 | 16px |
| 按钮 | 12px |
| 输入框 | 8px |
| 头像 | 50% (圆形) |

### 3.6 字体

- 主字体：系统字体栈 `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif`
- 标题：18-28px，粗体
- 正文：14-16px，常规
- 辅助文字：12-13px

### 3.7 间距

- 页面内边距：30px
- 卡片间距：20px
- 元素间距：15px

## 4. 粒子效果规范

### 4.1 Three.js 粒子系统

```javascript
// 粒子配置
{
    count: 2000,           // 粒子数量
    size: 0.5,             // 粒子大小
    color: '#4A90D9',      // 粒子颜色
    speed: 0.5,            // 移动速度
    interaction: true      // 鼠标交互
}
```

### 4.2 鼠标交互效果

- 鼠标移动时，附近粒子向鼠标方向散开
- 鼠标静止时，粒子缓慢聚集
- 粒子之间有连线效果（距离 < 100px 时连线）

## 5. 动画规范

### 5.1 过渡动画

```css
/* 淡入淡出 */
.fade-enter-active,
.fade-leave-active {
    transition: opacity 0.3s ease;
}
.fade-enter-from,
.fade-leave-to {
    opacity: 0;
}

/* 滑入 */
.slide-enter-active,
.slide-leave-active {
    transition: transform 0.3s ease;
}
.slide-enter-from {
    transform: translateX(30px);
}
.slide-leave-to {
    transform: translateX(-30px);
}
```

### 5.2 弹性效果 (Bounce)

```css
/* 按钮 hover */
.btn:hover {
    transform: scale(1.05);
    box-shadow: 0 0 20px rgba(0, 212, 255, 0.5);
}

.btn:active {
    transform: scale(0.95);
    animation: bounce 0.3s ease;
}

@keyframes bounce {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(0.95); }
    75% { transform: scale(1.02); }
}
```

### 5.3 卡片 hover

```css
.card:hover {
    transform: translateY(-5px);
    box-shadow: 0 12px 40px rgba(0, 212, 255, 0.2);
}
```
